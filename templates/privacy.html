Politique de confidentialité & RGPD

Responsable du traitement

L'hébergeur du Space Hugging Face et l'administrateur de l'instance agissent comme responsables du traitement des données de compte et d'usage. Pour toute demande : contact via l'administrateur du dépôt.

Données collectées

  • Compte: nom d'utilisateur, email, mot de passe (hashé bcrypt).
  • Scans: cibles, résultats JSON, horodatage, résumé IA optionnel.
  • Clés API tierces: stockées chiffrées (Fernet) dans Paramètres.
  • Graphe: entités, liens, scores de confiance pour vos investigations.
  • Cache API: réponses mises en cache (clé + TTL), sans données personnelles supplémentaires.
  • Rapports PDF: empreintes SHA-256 calculées à la volée ; le PDF n'est pas stocké par défaut.

Base légale

Les recherches portent sur des sources publiquement accessibles. Vous êtes responsable de l'usage conforme au RGPD (UE 2016/679), à la CNIL et aux lois locales. Ne traitez pas de données sensibles sans base légale explicite.

Vos droits (RGPD art. 15–22)

  • Accès: export JSON de vos scans via l'API ou l'historique.
  • Rectification: mise à jour du profil (email, mot de passe).
  • Effacement: suppression du compte et des scans associés — demande à l'administrateur.
  • Limitation / opposition: suspension des scans programmés ou du compte.
  • Portabilité: export CSV/JSON/PDF depuis Expert ou /api/v1/export.

Délai de réponse maximal : 30 jours(prolongation possible si demande complexe).

Suppression des données

Pour demander l'effacement de votre compte et de l'historique de scans : contactez l'administrateur avec votre identifiant utilisateur. Les caches API expirent automatiquement selon le TTL du connecteur.

Vérification d'intégrité des rapports

Chaque export PDF inclut des empreintes SHA-256 (en-têtes HTTP X-Document-Hash, X-Content-Hash). Vous pouvez vérifier un hash via GET /report/{id}/verify?hash=…(session connectée).
Vérification PDF :GET /verify/{id}— uploadez le PDF reçu ; l'empreinte SHA-256 est comparée à celle enregistrée lors de la génération officielle.

Rétention

Données hébergées sur Supabase (PostgreSQL). Cache API : TTL 24–168 h selon le connecteur.

Sécurité

HTTPS obligatoire sur Hugging Face. Ne partagez pas votre clé API REST. Secrets chiffrés au repos (Fernet).

Sous-traitants

  • Hugging Face (hébergement)
  • Supabase (base de données)
  • Groq (synthèse IA, si clé configurée)
  • Connecteurs tiers (Shodan, Hunter, etc. — selon vos clés)
