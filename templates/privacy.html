Politique de confidentialité & RGPD

Données collectées

  • Compte : nom d'utilisateur, email, mot de passe (hashé).
  • Scans : cibles recherchées, résultats JSON, horodatage (liés à votre compte si connecté).
  • Clés API : stockées chiffrées (Fernet) si vous les configurez dans Paramètres.
  • Entités & liens : graphe de corrélation pour vos investigations.

Base légale

Les recherches portent sur des sources publiquement accessibles. Vous êtes responsable de l'usage conforme au RGPD et aux lois locales.

Vos droits

Contactez l'administrateur du Space pour : accès, rectification, suppression de compte et export de vos données.

Rétention

Données hébergées sur Supabase (PostgreSQL). Cache API : TTL 24–168 h selon le connecteur.

Sécurité

HTTPS obligatoire sur Hugging Face. Ne partagez pas votre clé API REST.
